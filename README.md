# PROTO-FingerPrintTest

Compara dois arquivos de impressão digital em formato .PGM e retorna o valor de similaridade entre as impressões digitais. Usa a SourceAFIS para esse propósito. Um valor de similaridade de pelo menos 40 indica 99,99% de probabilidade de serem digitais da mesma pessoa. Abaixo disso, deve ser rejeitado.
